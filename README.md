# Multi-Host Ping Monitor

A Python script to monitor multiple hosts using ping, log results with timestamps, and display status (UP/DOWN) in real-time. Compatible with Windows, Linux, and macOS.

## Features

- Ping multiple hosts continuously
- Logs results with timestamp to a file (`ping_log.txt`)
- Displays status in the terminal
- Works on Windows, Linux, and macOS

## Requirements

- Python 3.x
